Class: paratii

paratii

new paratii(opts)

Paratii library main object The Paratii object serves as the general entry point for interacting with the family of Paratii contracts that are deployed on the blockchain, utilities to run and interact with a local IPFS node, and utilities to interact with the Paratii index.
Parameters:
Name Type Description
opts accountSchema options object to configure paratii library
Properties
Name Type Description
provider String optional - the address of an ethereum node (defaults to localhost:8754)
registryAddress String optional - the address where the Paratii Contract registry can be found
address String optional - address of the operator/user
privateKey String optional - private key of the user
