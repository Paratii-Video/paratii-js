Paratii

Paratii library main object The Paratii object serves as the general entry point for interacting with the family of Paratii contracts that are deployed on the blockchain, utilities to run and interact with a local IPFS node, and utilities to interact with the Paratii index.

new Paratii(opts: Object)
Parameters
opts (Object = {}) options object to configure paratii library
Name Description
opts.provider String optional - the address of an ethereum node (defaults to localhost:8754)
opts.registryAddress String optional - the address where the Paratii Contract registry can be found
opts.address String optional - address of the operator/user
opts.privateKey String optional - private key of the user
opts.ipfs Object TODO fix ipfs.repo --> ipfsrepo
opts.ipfs.repo String optional - namespace of the ipfs repository
opts.db Object TODO fix db.provider --> dbprovider
opts.db.provider String optional - baseURL of the mongoDb mirror
opts.mnemonic String optional - mnemonic of the user
Example
paratii = new Paratii({ 'eth.provider': 'http://localhost:8545', address: 'some-user-id', privateKey: 'some-user-priv-key'})
